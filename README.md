# Customer Churn Analysis using Power BI

## Objective
Analyze customer churn behavior and identify high-risk customers using Power BI dashboards and churn probability scoring.

## Tools Used
- Power BI
- DAX
- Excel/CSV Dataset

## Features
- Churn Probability Score
- Risk Segmentation
- Geography Analysis
- Gender Analysis
- Interactive Dashboard
